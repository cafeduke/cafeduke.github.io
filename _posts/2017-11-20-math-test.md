## Tex notation

### Inline
Quadratic equation $$ax^2 + bx + c = 0$$ factors are calculated using the following equation.

### Block
$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}.$$

## ASCII Math notation

### Inline
Quadratic equation `ax^2 + bx +  c = 0` factors are calculated using the following equation.

### Block
`x = (-b +- sqrt(b^2 - 4ac)) / (2a)`
